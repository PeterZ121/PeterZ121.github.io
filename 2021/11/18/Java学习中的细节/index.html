一个.java文件只能有一个public类，且该public类名与文件名一致

为什么？
每个编译单元（java文件）只能有一个public类，表示每个编译单元都有单一的公共接口。

常量池

在Java中，字符串被大量使用，为避免每次都创建相同的字符串对象和内存分配
在JVM内部用一块内存区域专门来存储字符串常量，该区域被称为常量池

    String str1 = "Hello";
    String str2 = "Hello";
    String str3 = "He"+"llo";

以上代码的结果是，str1,str2,str3都指向常量池中同一个字符串常量
